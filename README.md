# cvmbotJS
A CollabVM Bot written in Javascript

# How to run
1. Copy `config.example.toml` -> `config.toml`
2. Edit the configuration
3. Download required dependency. Do `npm install`
4. Run the bot. Do `node .` or `node index.js`
